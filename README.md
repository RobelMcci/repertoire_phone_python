# Projet gestion contact

# Besoin Fonctionnel
- Ajouter des contacts (nom, prénom, numéro portable)
- Consulter les contacts (rechercher par nom ou par numéro)
- Suprimer des contacts

# Besoin non fonctionnel
- Facilité de manipulation
- Usage friendly

# Objectifs
Concevoir un projet python avec une bonne structuration de projet, savoir bien séparer les models (contient les données), les services (contient la logique) et utiliser main.py comme point d’entrée du programme.
L'application n'a pas encore besoin d'interface graphique mais tout se passera dans la console (terminale). Toute utilisation de bibliothèque est recommander pour le devoir
